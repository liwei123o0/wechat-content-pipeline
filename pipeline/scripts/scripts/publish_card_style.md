# 知识卡片式排版（光音谷风格）

源自「光音谷piano」公众号的风格，适合教程/对比类文章。每节用浅灰底色卡片包裹，形成「知识卡片」阅读体验。

## 核心特征

- **每节独立卡片**：每个章节用 `background:#f7f6f4` 或 `#f5f5f4` 浅灰色背景卡片包裹，交替使用
- **中式序号标签**：用「零一｜核心逻辑」「零二｜Cursor」等标签（灰色14px）
- **H2标题紧凑**：17px bold，深蓝色 `#2c3e50`
- **正文色#333**：偏深对比度高
- **行距2.0**: 宽松舒适
- **两种记忆框交替**：隔节切换 A/B 型，避免视觉疲劳
  - A型：`border:1px solid #e8e8e8; background:#fafafa; border-radius:4px; text-align:center;`
  - B型：`border-left:3px solid #ccc; background:#ffffff; padding:14px 18px;`
- **🎯 实战建议**：emoji标记特定段落
- **表格渐变表头**：深蓝渐变 `#1a2a3a→#2c3e50`，金色文字 `#e8c35e`
- **红色金句框**：`border-left:3px solid #c0392b; color:#c0392b` 结尾情绪收束

## 颜色系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 标题卡片底色 | `#f0f5ec` + `#7db08a` 绿色顶条 + `#d4e0d0` 边框 | 绿色调开篇 |
| 引言卡片底色 | `#f7f8f9` + `#e8ecef` 边框 | 浅灰铺垫 |
| 章节卡片底色 | `#f7f6f4` / `#f5f5f4` | 暖灰交替 |
| 正文色 | `#333333` | 深灰 |
| 标题色 | `#2c3e50` | 深海蓝 |
| 标签色 | `#bbbbbb` | 小节编号 |
| 表格表头 | `linear-gradient(135deg,#1a2a3a,#2c3e50)` | 深蓝渐变 |
| 表格文字 | `#e8c35e`（表头）/ `#333`（表体） | 金色表头 |
| 金句框 | `border-left-color:#c0392b` + `color:#c0392b` | 红色 |
| 品牌署名 | `#888` 斜体居中 | 浅灰 |

## 字号层级

```
标题卡片主标题:  28px 粗体
章节标签:        14px 灰色
H2小标题:        17px 粗体
正文:            16px
记忆框文字:      15px
品牌署名:        15px 斜体
```

## 结构模板

```
[🟢 标题卡片] → 绿色调，28px大标题 + 灰色副标题
[⬜ 引言卡片] → 浅灰底，铺垫text
━━━━ 分隔线 ━━━━
[🟤 章节卡片 零一｜核心概念]
  ├─ 灰色标签「零一｜核心概念」
  ├─ H2标题
  ├─ 正文段落（定义）
  ├─ 正文章节（实例/场景）
  ├─ [记忆框A/B] → 一句话记住 / 核心用法
  └─ 🎯 实战建议
[🟤 章节卡片 零二｜...]
  ...（结构同上，记忆框交替类型）
━━━━ 分隔线 ━━━━
[🟤 速查卡片]
  ├─ 表格（深蓝渐变表头+金色文字）
  └─ [🔴 金句框] → 红色左边条收情绪
━━━━ 分隔线 ━━━━
[品牌署名] → 斜体居中灰色
```

## 适用场景

- 教程/知识科普类内容（4-6个并列知识点最佳）
- 工具对比、概念拆解、方法总结
- 每节结构统一、内容自包含的文章

## 发布方式

使用 `scripts/publish_card_style.py` 脚本：

```bash
cd /home/lw/wechat-content-pipeline/pipeline
python scripts/publish_card_style.py 创作/文章_xxx.md --account old
```

该脚本自动将标准 markdown 转换为知识卡片式内联 CSS HTML，并发布到微信草稿箱。
